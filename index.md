## 🎯 What Is Sandboxie Plus?

Sandboxie Plus creates a separate space on your computer called a sandbox. When you run a program inside this sandbox, it cannot change your system files, settings, or personal data. Think of it like a test kitchen. You can cook anything you want, but the mess stays inside the kitchen. Your real kitchen stays clean.

This tool helps you:
- Test new software without fear
- Open suspicious email attachments safely
- Browse risky websites without infecting your computer
- Run multiple versions of the same program
- Keep your privacy intact

## 🛡️ How to Use Sandboxie Plus

### Run a Program in a Sandbox

1. Right-click any program or file on your computer
2. Select "Run Sandboxed" from the menu
3. Choose which sandbox to use (DefaultBox works fine)
4. The program opens with a yellow border around its window

The yellow border tells you the program runs inside the sandbox. Any changes it makes stay inside the sandbox.

### Open a Website Safely

1. Open Sandboxie Plus
2. Right-click your default web browser in the program list
3. Select "Run Sandboxed"
4. Browse normally. Nothing you download or change will affect your real system.

### Check What Happens Inside

Open Sandboxie Plus while a sandboxed program runs. You will see files and registry changes the program created. This helps you understand what the program tried to do.

### Delete Sandbox Contents

When you finish testing, right-click the sandbox name in Sandboxie Plus. Select "Delete Content." This removes all files and changes made inside the sandbox. Your computer returns to its original state.

## ⚙️ Advanced Features

### Create Multiple Sandboxes

You can create different sandboxes for different tasks. For example:

- TestBox for new software
- WebBox for browsing

Right-click in the main window and select "Create New Sandbox." Give it a name and configure its settings.

### Set Sandbox Restrictions

Each sandbox has settings you can adjust. Right-click a sandbox and select "Sandbox Settings." You can:
- Block internet access for specific programs
- Limit how much disk space the sandbox uses
- Prevent programs from accessing your documents folder
- Set automatic deletion of sandbox contents on exit

### Use Quick Recovery

When you download a file inside a sandbox, you can recover it to your real system. Right-click the file in the sandbox and select "Quick Recovery." Choose where to save it on your real computer.

## ❓ Frequently Asked Questions

### Is Sandboxie Plus free?

Yes. Sandboxie Plus is free and open-source software. You can use it without paying anything.

### Does it work with all programs?

Most programs work inside a sandbox. Some programs with advanced hardware access may not work correctly. Try running the program in a sandbox first. If it fails, run it normally.

### Can I run games in a sandbox?

Yes, but games may run slower inside a sandbox. Games that need direct hardware access may not work at all.

### Will Sandboxie Plus slow down my computer?

The program uses very little system resources when idle. Running programs inside a sandbox may use slightly more memory and disk space than running them normally.
